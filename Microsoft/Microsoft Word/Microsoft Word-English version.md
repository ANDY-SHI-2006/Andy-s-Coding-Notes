## Chapter 0: Basics and Keyboard Shortcuts

### File Operations

| Operation | Shortcut/Method | Description |
|-----------|----------------|-------------|
| Quick Zoom | `Ctrl` + Scroll Wheel | Adjust document zoom level |
| Create New Document | `Ctrl` + `N` | Create a new blank Word document |
| Save As | - | Change format to solve compatibility issues between different versions |

### Editing Shortcuts

| Function | Shortcut | Description |
|----------|----------|-------------|
| Undo | `Ctrl` + `Z` | Undo the last action |
| Redo | `Ctrl` + `Y` | Restore the undone action |
| Repeat Last Action | `F4` or `Alt` + `Enter` | Can paste multiple times after copying, or press `F4` repeatedly |

> 💡 **Tip**: Use the clipboard to copy multiple times and paste all at once (clipboard content pastes from bottom to top)

### Alignment Shortcuts

| Alignment | Shortcut |
|-----------|----------|
| Center | `Ctrl` + `E` |
| Left Align | `Ctrl` + `L` |
| Right Align | `Ctrl` + `R` |

### Tab Key Functions

- **Indent First Line**: One `Tab` equals two spaces
- **Batch Tab**: Select multiple lines and press `Tab`
- **Reverse Tab**: `Shift` + `Tab`
- **In Tables**:
  - Press `Tab` to move right one cell
  - Press `Tab` in the bottom-right cell to add a new row automatically
- **Switch Between Documents**: `Ctrl` + `Tab`

### Selection Techniques

| Target | Operation |
|--------|-----------|
| Select a word | Double-click in the text |
| Select a sentence | Place cursor on the sentence, then `Ctrl` + Click |
| Select a line | Click on the left margin of the text box |
| Select a paragraph | Double-click on the left margin, or triple-click in the text |
| Continuous selection | Lock starting point with left click, hold `Shift` and click |
| Vertical selection | Hold `Alt` and drag with left mouse button |
| Add to selection (non-continuous) | Select first target, then hold `Ctrl` to add more |
| Select text with similar formatting | ① Select reference text ② Find "Select" icon in Home tab ③ Choose "Select Text with Similar Formatting" |
| Select all | Triple-click on left margin, or `Ctrl` + `A` |

### Special Keys

| Key | Function |
|-----|----------|
| `Backspace` | Delete to the left |
| `Del` | Delete to the right |
| `Home` | Go to beginning of line |
| `End` | Go to end of line |
| `PgUp` / `PgDn` | Scroll up/down one window (not one page of the document) |
| `Insert` (`Ins`) | When pressed, new text will replace existing text |

## Chapter 1: Home Tab

### Text Input and Format Painter

**Format Painter Usage**:
1. Select the format you want to copy, click Format Painter, then select the target text (suggest brushing from right to left)
2. Select original text then `Ctrl` + `Shift` + `C`, select target text then `Ctrl` + `Shift` + `V`

**Continuous Painting**:
- Double-click Format Painter after selecting to paint continuously
- To cancel: click Format Painter again / press `ESC`

**F4 Repeat Action**: After making changes, select other text and press `F4` to apply similar formatting

**Clear Formatting**:
- Eraser icon at the top
- Format Painter

### WPS Stacked Functions

Stacked functions in WPS:
1. Add Pinyin
2. Change Case (important function)
3. Add Enclosure (circle)
4. Add Border

### Change Case

Suitable for batch processing English text in multiple paragraphs

### Add Pinyin

1. Change font size first, then add Pinyin (otherwise it will be messy)
2. Keep only Pinyin: Right-click after selecting, click "Toggle Field Codes", delete corresponding Chinese characters to get Pinyin-only text

### Styles

### Bullets, Numbering, and Sorting

1. **Add Bullets**
2. **Promote/Demote Bullets**
3. Numbering is different from adding bullets
4. Press `Tab` after selecting to demote
5. Thesis writing can use numbering to achieve 1 → 1.1 → 1.11
6. **Sort**: Can sort first character by strokes, date, pinyin, or numbers in ascending/descending order

### Chinese Typography

2. **Merge Two Lines into One**: Achieve effect of text within parentheses
3. **Adjust Width**: Cooperate with centering to change character spacing
