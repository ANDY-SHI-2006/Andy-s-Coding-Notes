# 8 条件执行

## 8.1 关系与逻辑运算符

### 8.1.1 关系运算符

#### 8.1.1.1 小于（`<`）

若左操作数小于右操作数，返回 `true`。

```cpp
int a = 5, b = 10;
bool result = a < b;     // true（5 小于 10）
bool result2 = b < a;    // false（10 不小于 5）
```

#### 8.1.1.2 小于等于（`<=`）

若左操作数小于或等于右操作数，返回 `true`。

```cpp
int x = 5, y = 5, z = 3;
bool result1 = x <= y;   // true（5 等于 5）
bool result2 = z <= x;   // true（3 小于 5）
```

#### 8.1.1.3 大于（`>`）

若左操作数大于右操作数，返回 `true`。

```cpp
int a = 10, b = 5;
bool result = a > b;     // true（10 大于 5）
```

#### 8.1.1.4 大于等于（`>=`）

若左操作数大于或等于右操作数，返回 `true`。

```cpp
int x = 5, y = 5;
bool result = x >= y;    // true（5 等于 5）
```

#### 8.1.1.5 等于（`==`）

若两个操作数相等，返回 `true`。

```cpp
int a = 5, b = 5, c = 3;
bool result1 = a == b;   // true（5 等于 5）
bool result2 = a == c;   // false（5 不等于 3）
```

**浮点数比较的重要说明：**

由于精度误差，切勿直接使用 `==` 比较浮点数：

```cpp
// 错误——可能因精度问题而失败
if (x == 0.1) { }

// 正确——使用一个很小的 epsilon 值
const double EPSILON = 1e-9;
if (fabs(x - 0.1) < EPSILON) { }
```

#### 8.1.1.6 不等于（`!=`）

若操作数不相等，返回 `true`。

```cpp
int a = 5, b = 3;
bool result = a != b;    // true（5 不等于 3）
```

#### 8.1.1.7 关系运算符总结

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------ |
| `<` | 小于 | `5 < 10` | `true` |
| `<=` | 小于等于 | `5 <= 5` | `true` |
| `>` | 大于 | `10 > 5` | `true` |
| `>=` | 大于等于 | `5 >= 3` | `true` |
| `==` | 等于 | `5 == 5` | `true` |
| `!=` | 不等于 | `5 != 3` | `true` |

### 8.1.2 逻辑运算符

逻辑运算符组合布尔表达式。

#### 8.1.2.1 逻辑非（`!`）

否定布尔表达式。若操作数为 `false` 则返回 `true`，反之亦然。

```cpp
bool flag = true;
bool result = !flag;     // false

int x = 0;
if (!x) { }             // true，因为 x 是 0（假值）
```

#### 8.1.2.2 逻辑与（`&&`）

仅当**两个**操作数均为 `true` 时返回 `true`。

```cpp
int age = 25;
bool hasID = true;

if (age >= 18 && hasID) {
    // 两个条件必须都为真
    cout << "Entry allowed" << endl;
}
```

**`&&` 真值表：**

| A | B | A && B |
|---|---|--------|
| true | true | true |
| true | false | false |
| false | true | false |
| false | false | false |

#### 8.1.2.3 逻辑或（`\|\|`）

若**至少一个**操作数为 `true`，返回 `true`。

```cpp
bool isWeekend = true;
bool isHoliday = false;

if (isWeekend || isHoliday) {
    // 任一条件为真即可
    cout << "No work today" << endl;
}
```

**`\|\|` 真值表：**

| A | B | A \|\| B |
|---|---|----------|
| true | true | true |
| true | false | true |
| false | true | true |
| false | false | false |

#### 8.1.2.4 逻辑运算符总结

| 运算符 | 名称 | 描述 | 示例 |
|--------|------|------|------ |
| `!` | 逻辑非 | 否定表达式 | `!true` → `false` |
| `&&` | 逻辑与 | 两个操作数均为真时才为真 | `true && false` → `false` |
| `\|\|` | 逻辑或 | 至少一个操作数为真时才为真 | `true \|\| false` → `true` |

> **要点：**
> - 逻辑运算符也可以在条件内部使用。
> - 逻辑运算符比较的是条件，而非表达式。

### 8.1.3 运算符优先级

| 优先级 | 运算符 |
|--------|-------- |
| 最高 | `!`（逻辑非） |
| | 关系运算符：`<`、`<=`、`>`、`>=` |
| | 相等运算符：`==`、`!=` |
| | 逻辑与：`&&` |
| 最低 | 逻辑或：`\|\|` |


**推荐：** 使用括号使复杂条件清晰：

```cpp
// 不清晰——依赖优先级
if (a < b && c < d || e < f) { }

// 清晰——显式分组
if ((a < b && c < d) || e < f) { }
```

## 8.2 if 语句家族

本节涵盖 if 语句的各种形式以及条件执行的相关结构。

### 8.2.1 复合语句（块）

使用 `if`、`while`、`for` 等控制语句时，若需要执行**多条语句**（而非仅一条），必须将它们包裹在**花括号 `{}`** 中以形成一个**复合语句**（也称**块**）。

**单条语句**（不需要花括号）：
```cpp
if (x > 0)
    cout << "positive";  // 只有一条语句
```

> **⚠️ 警告：** 没有 `{}` 时，`if` 只控制**紧随其后的那条语句**。缩进不影响执行。

```cpp
if (x > 0)
    cout << "A";    // 只有这行属于 if
    cout << "B";    // 这行始终执行（编译器忽略缩进！）
```

**多条语句**（需要花括号形成复合语句）：
```cpp
if (x > 0) {              // 条件为真时执行
    cout << "positive";   // 语句 1
    x = 0;                // 语句 2
    count++;              // 语句 3
}                         // 复合语句结束
```

**要点：**

| 术语 | 含义 |
|------|------ |
| **复合语句** | 被视为单一单元的一组语句 |
| **块** | 复合语句的另一个名称 |
| **花括号** | 指花括号 `{}` |

- 在 C++ 中，任何需要"一条语句"的地方都可以用包裹在 `{}` 中的复合语句替换
- 复合语句可以包含任意数量的语句（0、1 或多条）
- 花括号定义变量的**作用域**——在花括号内定义的变量在花括号外不可访问

#### 8.2.1.1 空语句

**空语句**仅由一个分号 `;` 构成，没有表达式：

```cpp
;   // 这是一个有效的空语句
```

**重要：** 分号**不能**省略——没有它，编译器会报错：

```cpp
// 示例：if 中的空语句
if (x < 0)
    ;    // 空语句——当 x < 0 时不做任何事情
// ^ 这里的分号是必需的！

// 没有分号：错误
if (x < 0)
    // 错误：在下一行之前期望语句
```

> 另请参阅：[2.3.4 花括号风格](#234-brace-styles) 了解花括号放置位置的代码格式化约定。

### 8.2.2 if 语句

```cpp
if (condition) {
    // 仅在条件为真时执行的语句
}
```

### 8.2.3 if-else 语句

```cpp
if (condition) {
    // 条件为真时的语句
} else {
    // 条件为假时的语句
}
```

### 8.2.4 if-else-if 阶梯

```cpp
if (condition1) {
    // condition1 的语句
} else if (condition2) {
    // condition2 的语句
} else if (condition3) {
    // condition3 的语句
} else {
    // 以上都不满足时的语句
}
```

### 8.2.5 嵌套 if 语句

if 语句可以嵌套在其他 if 语句内部：

```cpp
if (condition1) {
    if (condition2) {
        // 两个条件都为真时的语句
    }
}
```

**悬垂 else 问题：**

`else` 总是绑定到最近的未匹配 `if`：

```cpp
if (a > 0)
    if (b > 0)
        cout << "Both positive" << endl;
else  // 这个 else 绑定到 if(b > 0)，而非 if(a > 0)！
    cout << "a is not positive" << endl;

// 使用花括号避免歧义：
if (a > 0) {
    if (b > 0) {
        cout << "Both positive" << endl;
    }
} else {
    cout << "a is not positive" << endl;
}
```

**最佳实践：** 即使只有一条语句，也始终对 if 语句使用花括号 `{}`，以防止错误并提高可读性。

> **注意：** `{}` 作为**代码块分隔符**——它明确定义了属于每个 `if`、`else` 或其他控制结构的语句块的边界。

### 8.2.6 条件（三元）运算符

简单 if-else 的紧凑形式。另请参阅 [4.8 三元条件运算符](#48-ternary-conditional-operator) 了解详细语法和优先级。

**语法：**

```cpp
condition ? expression_if_true : expression_if_false
```

**示例：**

```cpp
// 简单赋值
int max = (a > b) ? a : b;

// 在输出中使用
cout << (score >= 60 ? "Passed" : "Failed") << endl;

// 嵌套三元（谨慎使用——可能影响可读性）
int sign = (x > 0) ? 1 : (x < 0) ? -1 : 0;
```

**结合性：**

若一个表达式中有多个条件运算符，它们按**从右到左**结合。

```cpp
// 此表达式：
a ? b : c ? d : e

// 等价于（右结合）：
a ? b : (c ? d : e)

// 而非：
(a ? b : c) ? d : e
```

**示例：**
```cpp
int result = 0 ? 1 : 0 ? 2 : 3;  // 求值为：0 ? 1 : (0 ? 2 : 3)
                                 // 结果：3
```

## 8.3 switch 语句

`switch` 语句选择执行众多代码块中的一个。

### 8.3.1 switch 机制

当找到匹配的 `case` 时，执行**跳转到该 case**，然后**逐行**继续执行后续语句，**忽略其他 case 标签**，直到遇到 `break` 或 switch 块结束。

```cpp
int a = 1;
switch (a) {
    case 1:
        cout << "Case 1" << endl;   // 执行（匹配 a=1）
        // 没有 break！
    case 2:
        cout << "Case 2" << endl;   // 也执行（穿透）
        break;                       // 退出 switch
    default:
        cout << "Default" << endl;  // 不执行
}
// 输出：
// Case 1
// Case 2
```

**关键洞见：** `case` 只是一个**跳转标签**——它只告诉程序从哪里开始执行。没有 `break` 时，执行会顺序穿透所有后续 case（穿透行为）。

### 8.3.2 语法规则

### 8.3.2.1 控制表达式

`switch` 语句根据**控制表达式**选择要执行的语句，该表达式必须是**整型**（`int`、`char`、`enum` 等）。

```cpp
switch (expression) {  // 必须是整数或字符类型
    // case 标签...
}
```

**配合 `enum` 使用 `switch`：**

枚举常与 switch 语句一起使用，以获得更清晰、可读性更高的代码：

```cpp
enum class Color { Red, Green, Blue };

Color c = Color::Red;
switch (c) {
    case Color::Red:
        cout << "Red" << endl;
        break;
    case Color::Green:
        cout << "Green" << endl;
        break;
    case Color::Blue:
        cout << "Blue" << endl;
        break;
}
```

> **注意：** 使用 `enum class`（C++11）时，必须在 case 标签中使用作用域运算符（`Color::Red`）。

#### 8.3.2.2 case 标签必须是编译期常量

`case` 标签必须是**编译期常量**（不能是变量或在运行时求值的表达式）。

**有效的 case 标签：**

| 类型 | 示例 | 有效？ |
|------|------|-------- |
| 整数字面量 | `5`, `100` | ✓是 |
| 字符字面量 | `'a'`, `'A'` | ✓是 |
| 枚举值 | `Color::Red` | ✓是 |
| `const int` 常量 | `const int MAX = 10;` | ✓是（编译期常量） |
| 普通变量 | `int x = 5;` | ✗否 |
| 表达式 | `x + 1` | ✗否 |

**示例：**
```cpp
int x = 5;
switch (n) {
    case x:       // ✗错误：x 是变量
    case 5:       // ✓OK：字面量常量
    case 'A':     // ✓OK：字符常量
    case x + 1:   // ✗错误：表达式
}
```

> **原因：** 编译器为 switch 语句生成**跳转表**，这要求所有 case 值在编译期已知。

#### 8.3.2.3 case 标签必须唯一

`case` 标签必须是唯一的常量。**若两个或多个 case 标签具有相同的值，则会发生错误。**

```cpp
switch (n) {
    case 1: ...
    case 1: ...   // ✗错误：重复的 case 值
    case 'A': ... // ASCII 65
    case 65: ...  // ✗错误：'A' 等于 65，重复值
}
```

**来自不同来源的相同值也会导致错误：**

```cpp
const int MAX = 1;  // 编译期常量

switch (n) {
    case 1: ...      // 值 = 1
    case MAX: ...    // ✗错误：MAX 展开为 1，重复值
}
```

> **要点：** 编译器检查的是**最终数值**，而非 case 的写法。若两个 case 标签求值为相同的常量值，就是错误。

#### 8.3.2.4 break 语句

`break` 用于退出 switch 块。没有 `break` 时，执行会**穿透**到下一个 case。

```cpp
switch (choice) {
    case 1:
        cout << "One" << endl;
        break;      // ✓退出 switch
    case 2:
        cout << "Two" << endl;
        // 没有 break！执行继续到 case 3
    case 3:
        cout << "Three" << endl;
        break;
}
```

#### 8.3.2.5 default 标签

**default** 标签在没有 case 匹配时提供要执行的语句。它是**可选的**，但推荐用于处理意外值。

```cpp
switch (grade) {
    case 'A':
    case 'B':
    case 'C':
        cout << "Passed!" << endl;
        break;
    case 'D':
    case 'F':
        cout << "Failed!" << endl;
        break;
    default:           // 可选但推荐
        cout << "Invalid grade" << endl;
}
```

### 8.3.3 穿透行为

没有 `break` 时，执行会继续到下一个 case：

```cpp
int day = 3;
switch (day) {
    case 1:  // 周一
    case 2:  // 周二
    case 3:  // 周三
    case 4:  // 周四
    case 5:  // 周五
        cout << "Weekday" << endl;
        break;
    case 6:  // 周六
    case 7:  // 周日
        cout << "Weekend" << endl;
        break;
    default:
        cout << "Invalid day" << endl;
}
// 输出：Weekday
```

### 8.3.4 有意穿透

有时你可能有意省略 `break`，让多个 `case` 标签共享同一个代码块：

```cpp
switch (grade) {
    case 'A':
    case 'B':
    case 'C':
        cout << "Passed!" << endl;  // A、B、C 都共享此输出
        break;
    case 'D':
    case 'F':
        cout << "Failed!" << endl;  // D、F 共享此输出
        break;
}
```

> **注意：** 这种"穿透"行为有时是有意设计的，但在大多数情况下，它是忘记写 `break` 导致的 bug。

### 8.3.5 switch 与 if-else 对比

| 使用 `switch` | 使用 `if-else` |
|--------------|--------------- |
| 测试单个变量 | 复杂条件 |
| 整数或字符值 | 浮点数值 |
| 多个离散值 | 值的范围 |
| 仅相等检查 | 关系比较 |

## 8.4 将值用作条件

在 C++ 中，单个值可以直接用作条件，无需关系运算符：

```cpp
int a = 5;
if (a) {
    count++;  // 执行，因为 a 非零
}
```

**真值规则：**

| 值 | 解释为 | 示例 |
|----|--------|------ |
| 零（`0`、`0.0`、`\0`） | `false` | `if (0)` → 条件为假 |
| 非零（任何非零值） | `true` | `if (5)` → 条件为真 |

**常见使用场景：**

```cpp
// 检查指针是否非空
if (ptr) { }           // 等价于：if (ptr != nullptr)

// 检查字符是否非空终止符
if (ch) { }            // 等价于：if (ch != '\0')

// 检查整数是否非零
if (count) { }         // 等价于：if (count != 0)
```

## 8.5 循环结构

### 8.5.1 while 循环

每次迭代前测试条件：

```cpp
while (condition) {
    // 条件为真时执行的语句
}
```

**示例：**

```cpp
int count = 0;
while (count < 5) {
    cout << count << " ";
    count++;
}
// 输出：0 1 2 3 4
```

**特性：**

- 条件在**每次迭代前**检查
- 若条件初始为假，可能执行零次
- 循环变量必须在循环前初始化
- 循环变量必须在循环内更新

### 8.5.2 do-while 循环

每次迭代后测试条件（保证至少执行一次）：

```cpp
do {
    // 至少执行一次的语句
} while (condition);
```

**示例：**

```cpp
int num;
do {
    cout << "Enter a positive number: ";
    cin >> num;
} while (num <= 0);
// 用户至少被提示一次
```

**对比：**

| 方面 | `while` | `do-while` |
|------|---------|------------ |
| 条件检查 | 迭代前 | 迭代后 |
| 最少执行次数 | 0 | 1 |
| 使用场景 | 可能不需要迭代时 | 至少需要一次迭代时 |

### 8.5.3 for 循环

在一行中紧凑地包含初始化、条件和更新的循环。

#### 8.5.3.1 基本语法与执行顺序

```cpp
for (initialization; condition; update) {
    // statements
}
```

**执行顺序：**

1. **初始化**（一次，在开始处）
2. 检查**条件** → 若为假，退出循环
3. 执行**循环体**
4. 执行**更新**
5. 回到第 2 步

**基本示例：**

```cpp
for (int i = 0; i < 5; i++) {
    cout << i << " ";
}
// 输出：0 1 2 3 4
```

#### 8.5.3.2 初始化选项

**初始化部分是可选的。** 三种方法：

| 方法 | 语法 | 变量作用域 | 使用场景 |
|------|------|-----------|---------- |
| **循环内（推荐）** | `for (int i = 0; ...)` | 仅在循环内 | **首选**——限制作用域，防止命名冲突 |
| **循环外** | `int i = 0; for (; ...)` | 循环外可访问 | 需要在循环结束后使用计数器值 |
| **多个变量** | `for (int i = 0, j = 10; ...)` | 仅在循环内 | 多个计数器 |

**示例——循环内（推荐）：**

```cpp
for (int i = 0; i < 5; i++) {
    cout << i;
}
// i 在此处不再存在（作用域限制在循环内）
```

**示例——循环外：**

```cpp
int i = 0;
for (; i < 5; i++) {          // 初始化留空，分号必需
    cout << i << " ";
}
// i 在此处仍然存在，值为 5
```

**示例——多个变量：**

```cpp
for (int i = 0, j = 10; i < j; i++, j--) {
    cout << i << "," << j << " ";
}
// 输出：0,10 1,9 2,8 3,7 4,6
```

> **最佳实践：** 尽可能在 for 循环内部定义变量。这限制了变量作用域，减少了命名冲突，并防止意外修改。

#### 8.5.3.3 特殊循环模式

**省略部分：**

三个部分中的任何一个都可以省略（很少使用）：

```cpp
int i = 0;
for (; i < 5; ) {      // 省略初始化和更新
    cout << i++ << " ";
}
```

**无限循环：**

```cpp
for (;;) {             // 所有部分省略，等价于 while(true)
    // 无限循环——使用 break 退出
}
```

#### 8.5.3.4 基于范围的 for 循环（C++11）

用于不显式使用下标遍历容器。

**语法：**

```cpp
for (element_declaration : container) {
    // statements
}
```

**按值（拷贝）：**

创建每个元素的副本。适用于 `int`、`char`、`bool` 等小类型。

```cpp
vector<int> nums = {1, 2, 3, 4, 5};

for (int n : nums) {
    cout << n << " ";
}
// n 是副本——修改不影响原始值
```

**按引用（修改原始值）：**

```cpp
for (int& n : nums) {
    n *= 2;  // 将原始 vector 中每个元素翻倍
}
```

**按 const 引用（只读，高效）：**

避免复制大对象，同时防止修改。

```cpp
for (const int& n : nums) {
    cout << n << " ";
}
```

**使用 Auto（最常见的形式）：**

```cpp
// 修改版本
for (auto& n : nums) {
    n *= 2;
}

// 只读版本（推荐的默认形式）
for (const auto& n : nums) {
    cout << n << " ";
}

// 拷贝版本（需要副本时）
for (auto n : nums) {
    // n 是副本——安全修改而不影响原始值
}
```

> **最佳实践：** 使用基于范围的 for 循环进行简单迭代——更简洁且不易出错。只读访问默认使用 `const auto&`，修改时使用 `auto&`。

### 8.5.4 嵌套循环

循环可以嵌套在其他循环内部：

```cpp
// 打印乘法表
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= 5; j++) {
        cout << i * j << "\t";
    }
    cout << endl;
}
```

**输出：**

```
1       2       3       4       5
2       4       6       8       10
3       6       9       12      15
4       8       12      16      20
5       10      15      20      25
```

## 8.6 跳转语句

### 8.6.1 break 语句

`break` 立即退出最近的封闭循环或 switch：

```cpp
// 提前退出循环
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;  // i 达到 5 时退出循环
    }
    cout << i << " ";
}
// 输出：0 1 2 3 4
```

```cpp
// 退出 switch case（防止穿透）
switch (choice) {
    case 1:
        cout << "Option 1" << endl;
        break;  // 没有 break，执行会继续到 case 2
    case 2:
        cout << "Option 2" << endl;
        break;
}
```

### 8.6.2 continue 语句

`continue` 跳过当前迭代的剩余部分，继续下一次迭代：

```cpp
// 只打印奇数
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue;  // 跳过偶数
    }
    cout << i << " ";
}
// 输出：1 3 5 7 9
```

**对比：**

| 语句 | 效果 |
|------|------ |
| `break` | 立即退出循环/switch |
| `continue` | 跳到循环的下一次迭代 |

## 8.7 循环对比与选择

| 循环类型 | 最适合 | 关键特性 |
|----------|--------|---------- |
| `while` | 未知迭代次数，由条件驱动 | 先检查条件 |
| `do-while` | 至少需要一次迭代 | 后检查条件 |
| `for` | 已知迭代次数，计数 | 带计数器的紧凑语法 |
| 范围 for | 遍历集合 | 容器最简洁 |
