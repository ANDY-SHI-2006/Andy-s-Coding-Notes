# 12 高级主题

## 12.1 指针进阶

### 12.1.1 指针与 const

| 声明 | 含义 | 能否修改指向的值？ | 能否改变指针？ |
|------|------|-------------------|---------------|
| `const int* p` | 指向常量的指针 | 否 | 是 |
| `int* const p` | 常量指针 | 是 | 否 |
| `const int* const p` | 指向常量的常量指针 | 否 | 否 |

```cpp
const int* p = &a;  // 不能修改 *p，但可以改变 p
int* const q = &b;  // 可以修改 *q，但不能改变 q
```

### 12.1.2 指针与数组

- 数组名是指向首元素的常量指针
- `arr[i]` 等价于 `*(arr + i)`

```cpp
int arr[5] = {1, 2, 3, 4, 5};
int* ptr = arr;  // 等同于 &arr[0]

// 等价的访问方式：
cout << arr[2];     // 3
cout << *(arr + 2); // 3
cout << ptr[2];     // 3
cout << *(ptr + 2); // 3
```

**指针算术运算：**
```cpp
int* p = arr;
p++;      // 移动到下一个整数（增加 sizeof(int)）
p--;      // 移动到前一个整数
p + 3;    // 指向前面第 3 个位置的元素
```

### 12.1.3 指向结构体的指针

```cpp
struct Person {
    string name;
    int age;
};

Person p = {"John", 25};
Person* ptr = &p;

// 访问成员：
cout << (*ptr).age;   // 25
cout << ptr->age;     // 25（箭头运算符）
```

## 12.2 动态内存管理

### 12.2.1 new 运算符

```cpp
// 单个元素
int* p = new int;       // 为一个 int 分配内存
*p = 123;

// 数组
int size;
cin >> size;
int* arr = new int[size];  // 运行时确定大小

// 结构体
Person* p = new Person;
p->age = 25;
```

### 12.2.2 delete 运算符

```cpp
// 单个元素
delete p;
p = nullptr;  // 良好实践

// 数组
delete[] arr;
```

### 12.2.3 内存泄漏

- 分配的内存未释放时发生
- 仅丢失了存储地址的指针

```cpp
int main() {
    int* q = new int;  // 分配
    q = p;             // 丢失了新分配的内存地址！
    // 内存泄漏：无法释放原始分配的内存
}
```

> **最佳实践：** `new` 与 `delete` 必须成对使用，删除后将指针置为 `nullptr`。

## 12.3 引用

**概念：** 变量的别名

```cpp
int x = 456;
int& intRef = x;  // intRef 是 x 的别名

intRef++;         // x 现在为 457
```

**特性：**
- 声明时必须初始化
- 不能为 null
- 不能重新绑定到另一个变量

### 12.3.1 按引用传递

```cpp
void swap(int& a, int& b) {  // 按引用传递
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 5, y = 7;
    swap(x, y);  // x 和 y 实际被交换了
}
```

> **建议：** 在可能的情况下使用引用代替指针——更安全、语法更简洁。
