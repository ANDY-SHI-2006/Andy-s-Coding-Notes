# 11 标准模板库

标准模板库（STL）提供了一系列模板类和函数，用于常见的数据结构与算法。`vector` 是最常用的 STL 容器。


## 11.0 标准模板库（STL）概览

标准模板库（STL）是 C++ 标准库的重要组成部分，提供了可复用的泛型容器类和算法。


### 11.0.1 STL 组件

STL 由三大核心组件构成：

1. **容器（Containers）** —— 用于存储对象的数据结构
2. **迭代器（Iterators）** —— 用于遍历容器的对象
3. **算法（Algorithms）** —— 通过迭代器对容器进行操作的函数

**关系：**
```
算法 ←→ 迭代器 ←→ 容器
```


#### 8.3.2 STL 容器

容器是用于存储和管理对象集合的模板类。

**通用容器方法（适用于所有容器）：**

| 方法 | 说明 |
|------|------|
| `size()` | 元素个数 |
| `empty()` | 检查容器是否为空 |
| `clear()` | 移除所有元素 |
| `swap(c)` | 与另一个容器交换内容 |

**容器分类：**

| 类别 | 容器 | 特性 |
|------|------|------|
| **序列容器** | `vector`、`deque`、`list`、`forward_list`、`array` | 线性顺序 |
| **关联容器** | `set`、`multiset`、`map`、`multimap` | 有序，基于键 |
| **无序容器** | `unordered_set`、`unordered_map` | 基于哈希，O(1) |
| **容器适配器** | `stack`、`queue`、`priority_queue` | 受限接口 |


### 11.0.3 Vector（`<vector>`）

**特性：**
- 动态数组（大小可变）
- 连续内存存储
- 随机访问速度快：O(1)
- 尾部插入速度快：均摊 O(1)
- 中部插入/删除速度慢：O(n)

**声明：**
```cpp
#include <vector>
using namespace std;

vector<int> v1;              // 空 vector
vector<int> v2(10);          // 10 个元素（值初始化）
vector<int> v3(10, 5);       // 10 个元素，均为 5
vector<int> v4 = {1, 2, 3};  // 初始化列表
vector<int> v5(v4);          // 拷贝构造函数
```

**常用方法：**

| 方法 | 说明 | 复杂度 |
|------|------|--------|
| `push_back(x)` | 在末尾添加元素 | 均摊 O(1) |
| `pop_back()` | 移除末尾元素 | O(1) |
| `front()` | 首元素 | O(1) |
| `back()` | 末元素 | O(1) |
| `at(i)` | 访问第 i 个元素（带边界检查） | O(1) |
| `operator[i]` | 访问第 i 个元素（不检查） | O(1) |
| `insert(pos, x)` | 在指定位置插入 | O(n) |
| `erase(pos)` | 移除指定位置元素 | O(n) |
| `resize(n)` | 改变大小 | O(n) |
| `capacity()` | 当前容量 | O(1) |
| `reserve(n)` | 为 n 个元素预分配空间 | O(n) |

**示例：**
```cpp
vector<int> v;
v.push_back(10);
v.push_back(20);
v.push_back(30);

cout << v.size();     // 3
cout << v[0];         // 10
cout << v.front();    // 10
cout << v.back();     // 30

v.pop_back();         // 移除 30
v.insert(v.begin() + 1, 15);  // 在位置 1 插入 15
v.erase(v.begin());   // 移除首元素
```


### 11.0.4 迭代器

迭代器是指针的抽象，用于遍历容器。

**迭代器类型：**

| 迭代器 | 能力 | 支持的容器 |
|--------|------|-----------|
| 输入迭代器 | 只读，正向 | istream |
| 输出迭代器 | 只写，正向 | ostream |
| 正向迭代器 | 读/写，正向 | forward_list |
| 双向迭代器 | 读/写，正向/反向 | list、set、map |
| 随机访问迭代器 | 读/写，随机访问 | vector、deque、array |

**迭代器声明：**
```cpp
vector<int>::iterator it;           // 正向迭代器
vector<int>::reverse_iterator rit;  // 反向迭代器
```

**常用迭代器操作：**

| 操作 | 说明 |
|------|------|
| `*it` | 解引用（访问元素） |
| `it++` | 移到下一个元素 |
| `it--` | 移到上一个元素（双向迭代器） |
| `it + n` | 前进 n 个位置（随机访问） |
| `it - n` | 后退 n 个位置（随机访问） |
| `it1 - it2` | 两个迭代器之间的距离 |

**容器迭代器方法：**

| 方法 | 返回值 |
|------|--------|
| `begin()` | 指向首元素的迭代器 |
| `end()` | 指向尾后位置的迭代器 |
| `rbegin()` | 反向迭代器，指向末元素 |
| `rend()` | 反向迭代器，指向首前位置 |
| `cbegin()` | const 迭代器（C++11） |

**迭代器示例：**
```cpp
vector<int> v = {10, 20, 30, 40, 50};

// 正向遍历
for (vector<int>::iterator it = v.begin(); it != v.end(); ++it) {
    cout << *it << " ";  // 10 20 30 40 50
}

// 反向遍历
for (vector<int>::reverse_iterator rit = v.rbegin(); rit != v.rend(); ++rit) {
    cout << *rit << " ";  // 50 40 30 20 10
}

// 基于范围的 for 循环（C++11）——更简洁！
for (int x : v) {
    cout << x << " ";
}
```

**⚠️ 迭代器失效：**
- `vector`：插入/删除可能导致所有迭代器失效
- `list`：仅被删除元素的迭代器失效
- 修改容器后务必“重置”迭代器


### 11.0.5 STL 算法（`<algorithm>`）

STL 提供了与迭代器配合使用的泛型算法。

**常用算法：**

| 算法 | 说明 |
|------|------|
| `sort(begin, end)` | 排序元素 |
| `find(begin, end, val)` | 查找值 |
| `binary_search(begin, end, val)` | 二分查找（需有序区间） |
| `count(begin, end, val)` | 统计出现次数 |
| `min_element(begin, end)` | 查找最小值 |
| `max_element(begin, end)` | 查找最大值 |
| `reverse(begin, end)` | 反转元素 |
| `copy(srcBegin, srcEnd, dstBegin)` | 拷贝区间 |
| `fill(begin, end, val)` | 用值填充 |
| `accumulate(begin, end, init)` | 累加元素（`<numeric>`） |

**示例：**
```cpp
#include <algorithm>
#include <numeric>
#include <vector>

vector<int> v = {30, 10, 50, 20, 40};

// 排序
sort(v.begin(), v.end());  // 10, 20, 30, 40, 50

// 查找
auto it = find(v.begin(), v.end(), 30);
if (it != v.end()) {
    cout << "在位置 " << (it - v.begin()) << " 找到";
}

// 最小/最大值
auto minIt = min_element(v.begin(), v.end());
auto maxIt = max_element(v.begin(), v.end());

// 统计
int count30 = count(v.begin(), v.end(), 30);

// 累加
int sum = accumulate(v.begin(), v.end(), 0);  // <numeric>

// 反转
reverse(v.begin(), v.end());

// 使用自定义比较器排序
sort(v.begin(), v.end(), greater<int>());  // 降序
```


### 11.0.6 容器对比

| 容器 | 访问 | 插入/删除 | 适用场景 |
|------|------|-----------|----------|
| `vector` | O(1) 随机访问 | O(1) 尾部，O(n) 中部 | 默认选择 |
| `list` | O(n) 顺序访问 | O(1) 任意位置 | 频繁插入删除 |
| `deque` | O(1) 随机访问 | O(1) 两端 | 双端队列 |
| `set` | O(log n) | O(log n) | 有序唯一元素 |
| `map` | O(log n) | O(log n) | 键值对 |
| `unordered_map` | O(1) 平均 | O(1) 平均 | 快速查找 |


### 11.0.7 最佳实践

1. **默认使用 `vector`** —— 大多数情况下效率最高
2. 大小已知时**预分配容量**：`v.reserve(1000);`
3. 为了可读性，**优先使用基于范围的 for 循环**
4. 修改容器后**检查迭代器有效性**
5. **使用算法**代替手写循环


## 11.1 Vector

`vector` 是一种**动态数组**，可以自动增长或缩小。与固定大小的数组不同，vector 在内部处理内存管理，并提供丰富的操作接口。

### 11.1.1 为何使用 Vector？

**固定大小数组的问题：**
```cpp
int arr[100];  // 固定大小，无法在运行时改变
// 如果需要 101 个元素怎么办？或者只需要 10 个？
```

**使用 vector 的解决方案：**
```cpp
vector<int> v;  // 空 vector，可动态增长
v.push_back(10);  // 按需添加元素
v.push_back(20);  // 自动管理内存
```

**核心优势：**

| 特性 | 数组 | Vector |
|------|------|--------|
| **大小** | 编译时固定 | 运行时动态增减 |
| **内存** | 手动管理 | 自动内存管理 |
| **安全性** | 无边界检查 | `at()` 提供边界检查 |
| **操作** | 有限（原始指针） | 丰富的内置方法 |
| **灵活性** | 不可调整大小 | 可调整大小、插入、删除 |

### 11.1.2 头文件与声明

```cpp
#include <vector>  // 必需的头文件

using namespace std;  // 或显式使用 std::vector

// 声明
vector<int> v1;              // 整型空 vector
vector<double> v2;           // 双精度浮点型空 vector
vector<string> v3;           // 字符串空 vector
vector<vector<int>> matrix;  // 二维 vector（vector 的 vector）
```

### 11.1.3 初始化

**1. 空 vector：**
```cpp
vector<int> v;  // 大小 = 0，容量 = 0
```

**2. 指定初始大小（以默认值填充）：**
```cpp
vector<int> v(5);        // 5 个元素，均为 0
vector<string> s(3);     // 3 个元素，均为空字符串
```

**3. 指定初始大小和值：**
```cpp
vector<int> v(5, 100);        // 5 个元素，均为 100：{100, 100, 100, 100, 100}
vector<string> s(3, "hi");    // 3 个元素，均为 "hi"
```

**4. 列表初始化（C++11）：**
```cpp
vector<int> v = {1, 2, 3, 4, 5};  // 5 个元素
vector<int> v{1, 2, 3, 4, 5};     // 同上
vector<string> colors{"red", "green", "blue"};
```

**5. 从另一个 vector 拷贝：**
```cpp
vector<int> v1 = {1, 2, 3};
vector<int> v2(v1);           // 拷贝构造函数
vector<int> v3 = v1;          // 拷贝赋值
vector<int> v4(v1.begin(), v1.end());  // 使用迭代器拷贝
```

**6. 从数组初始化：**
```cpp
int arr[] = {1, 2, 3, 4, 5};
vector<int> v(arr, arr + 5);  // 从数组拷贝
```

> **注意：** 使用特定值初始化时，优先使用列表初始化 `{}` 以提高清晰度。

### 11.1.4 常用操作

| 操作 | 说明 | 示例 | 时间复杂度 |
|------|------|------|------------|
| `push_back(x)` | 在末尾添加元素 | `v.push_back(10)` | 均摊 O(1) |
| `pop_back()` | 移除末尾元素 | `v.pop_back()` | O(1) |
| `insert(pos, x)` | 在指定位置插入 | `v.insert(v.begin(), 5)` | O(n) |
| `erase(pos)` | 移除指定位置元素 | `v.erase(v.begin())` | O(n) |
| `clear()` | 移除所有元素 | `v.clear()` | O(n) |
| `empty()` | 检查是否为空 | `if (v.empty())` | O(1) |
| `size()` | 元素个数 | `int n = v.size()` | O(1) |
| `capacity()` | 存储容量 | `int c = v.capacity()` | O(1) |

**示例：**
```cpp
vector<int> v = {1, 2, 3};

v.push_back(4);      // v = {1, 2, 3, 4}
v.pop_back();        // v = {1, 2, 3}
v.insert(v.begin(), 0);  // v = {0, 1, 2, 3}
v.erase(v.begin() + 1);  // v = {0, 2, 3}

if (!v.empty()) {
    cout << "大小: " << v.size() << endl;  // 3
}
```

### 11.1.5 元素访问

| 方法 | 说明 | 示例 | 备注 |
|------|------|------|------|
| `v[i]` | 下标运算符 | `v[0]` | 无边界检查 |
| `v.at(i)` | 带检查的访问 | `v.at(0)` | 越界时抛出异常 |
| `v.front()` | 首元素 | `v.front()` | 等同于 `v[0]` |
| `v.back()` | 末元素 | `v.back()` | 等同于 `v[v.size()-1]` |
| `v.data()` | 原始指针 | `int* p = v.data()` | 兼容 C 风格 API |

**示例：**
```cpp
vector<int> v = {10, 20, 30, 40, 50};

cout << v[0] << endl;       // 10（无边界检查）
cout << v.at(0) << endl;    // 10（带边界检查）
cout << v.front() << endl;  // 10
cout << v.back() << endl;   // 50

// 使用 at() 安全访问
try {
    cout << v.at(100) << endl;  // 抛出 std::out_of_range
} catch (const std::out_of_range& e) {
    cout << "错误: " << e.what() << endl;
}
```

> **建议：** 需要边界检查（安全性）时使用 `at()`，确定索引无误（性能）时使用 `[]`。

### 11.1.6 大小与容量

| 方法 | 说明 | 示例 |
|------|------|------|
| `size()` | 元素个数 | `v.size()` |
| `capacity()` | 已分配空间（不重新分配即可容纳的元素数） | `v.capacity()` |
| `resize(n)` | 将大小调整为 n | `v.resize(10)` |
| `resize(n, val)` | 调整大小并用 val 填充新元素 | `v.resize(10, 0)` |
| `reserve(n)` | 为至少 n 个元素预分配空间 | `v.reserve(100)` |
| `shrink_to_fit()` | 将容量缩减至大小 | `v.shrink_to_fit()` |

**大小与容量的区别：**
```cpp
vector<int> v;
cout << v.size() << " " << v.capacity() << endl;  // 0 0

v.push_back(1);  // 大小: 1，容量: 1
v.push_back(2);  // 大小: 2，容量: 2
v.push_back(3);  // 大小: 3，容量: 4（通常翻倍）

v.reserve(100);  // 容量 >= 100，大小不变
cout << v.size() << " " << v.capacity() << endl;  // 3 100

v.resize(5);     // 大小: 5，新元素默认初始化
v.resize(10, 99); // 大小: 10，新元素为 99
```

> **注意：** 当 `size()` 超过 `capacity()` 时，vector 会自动重新分配内存（通常容量翻倍）。这会导致迭代器和引用失效。

### 11.1.7 迭代器

迭代器提供了统一的方式来遍历容器。

| 迭代器 | 说明 | 示例 |
|--------|------|------|
| `begin()` | 指向首元素的迭代器 | `auto it = v.begin()` |
| `end()` | 指向尾后位置的迭代器 | `auto it = v.end()` |
| `rbegin()` | 反向迭代器，指向末元素 | `auto rit = v.rbegin()` |
| `rend()` | 反向迭代器，指向首前位置 | `auto rit = v.rend()` |
| `cbegin()` | const 迭代器（C++11） | `auto cit = v.cbegin()` |
| `cend()` | const 迭代器（C++11） | `auto cit = v.cend()` |

**使用迭代器：**
```cpp
vector<int> v = {10, 20, 30, 40, 50};

// 正向遍历
for (auto it = v.begin(); it != v.end(); ++it) {
    cout << *it << " ";  // 解引用迭代器
}
// 输出: 10 20 30 40 50

// 反向遍历
for (auto rit = v.rbegin(); rit != v.rend(); ++rit) {
    cout << *rit << " ";
}
// 输出: 50 40 30 20 10

// 基于范围的 for 循环（C++11）——最常用
for (const auto& elem : v) {
    cout << elem << " ";
}

// 使用索引
for (size_t i = 0; i < v.size(); i++) {
    cout << v[i] << " ";
}
```

> **最佳实践：** 简单遍历时使用基于范围的 for 循环（`for (auto& elem : v)`）——更简洁且不易出错。

### 11.1.8 配合 Vector 使用算法

STL 算法通过迭代器与 vector 无缝协作。

```cpp
#include <vector>
#include <algorithm>  // 算法头文件
#include <numeric>    // 累加头文件

vector<int> v = {3, 1, 4, 1, 5, 9, 2, 6};

// 排序
sort(v.begin(), v.end());                    // 升序: {1, 1, 2, 3, 4, 5, 6, 9}
sort(v.begin(), v.end(), greater<int>());    // 降序

// 查找
if (binary_search(v.begin(), v.end(), 5)) {
    cout << "找到 5!" << endl;
}

auto it = find(v.begin(), v.end(), 5);       // 查找元素
if (it != v.end()) {
    cout << "在索引 " << (it - v.begin()) << " 找到" << endl;
}

// 最小/最大值
auto minIt = min_element(v.begin(), v.end());
auto maxIt = max_element(v.begin(), v.end());

// 反转
reverse(v.begin(), v.end());

// 累加（求和）
int sum = accumulate(v.begin(), v.end(), 0);

// 统计出现次数
int count1 = count(v.begin(), v.end(), 1);

// remove-erase 惯用法
v.erase(remove(v.begin(), v.end(), 1), v.end());  // 移除所有 1
```

### 11.1.9 Vector 与数组的对比

| 特性 | 数组 (`int arr[]`) | Vector (`vector<int>`) |
|------|---------------------|------------------------|
| **大小** | 编译时固定 | 运行时动态 |
| **内存** | 栈（通常） | 堆 |
| **调整大小** | 不可行 | `resize()`、`push_back()` |
| **边界检查** | 无 | 有（`at()`） |
| **内存管理** | 自动（栈） | 自动（RAII） |
| **拷贝/赋值** | 不可赋值 | 完全可拷贝、可赋值 |
| **传入函数** | 退化为指针 | 保持大小信息 |
| **开销** | 无 | 很小（大小、容量指针） |
| **性能** | 最优 | 略慢（可忽略） |

**何时使用数组：**
- 编译时大小固定且已知
- 性能至关重要
- 嵌入式系统资源受限
- 需要 C 语言兼容

**何时使用 Vector：**
- 编译时大小未知
- 需要动态增删元素
- 安全性与便利性优先
- 使用 STL 算法

### 11.1.10 最佳实践

**1. 优先使用 `emplace_back` 而非 `push_back`（C++11）：**
```cpp
vector<pair<int, string>> v;
v.push_back(make_pair(1, "hello"));  // 创建临时对象
v.emplace_back(1, "hello");          // 原地构造，更高效
```

**2. 大小已知时预分配容量：**
```cpp
vector<int> v;
v.reserve(10000);  // 避免多次重新分配
for (int i = 0; i < 10000; i++) {
    v.push_back(i);  // 10000 个元素内不会重新分配
}
```

**3. 使用 `empty()` 而非 `size() == 0`：**
```cpp
// 推荐：
if (!v.empty()) { }

// 避免：
if (v.size() > 0) { }  // 可用但不够地道
```

**4. 传参时使用 const 引用避免拷贝：**
```cpp
// 高效 —— 不拷贝
void printVector(const vector<int>& v) {
    for (const auto& elem : v) {
        cout << elem << " ";
    }
}

// 低效 —— 拷贝整个 vector
void badPrint(vector<int> v) {  // 不要这样做
    // ...
}
```

**5. 注意迭代器失效：**
```cpp
vector<int> v = {1, 2, 3, 4, 5};
auto it = v.begin() + 2;  // 指向 3

v.push_back(6);  // 可能导致所有迭代器失效！
// *it 现在是未定义行为
```

> **警告：** 插入操作（`push_back`、`insert`）在发生重新分配时可能导致迭代器失效。尽可能使用 `reserve()` 来避免。

## 11.2 String

`std::string` 是一个类模板，表示字符序列。它提供了一个可动态调整大小的字符数组，并带有许多便捷的字符串操作成员函数。

### 11.2.1 为何使用 string？

**C 风格字符串（char 数组）的问题：**
```cpp
char str[20] = "hello";
// - 固定大小，无法增长
// - strcpy、strcat 很危险（缓冲区溢出）
// - 手动内存管理
// - 不能用 == 比较，必须用 strcmp
```

**使用 string 的解决方案：**
```cpp
string s = "hello";
// - 动态大小，自动增长
// - 带边界检查的安全操作
// - 自动内存管理
// - 用 == 比较，用 + 连接
```

**核心优势：**

| 特性 | C 字符串 (`char[]`) | `std::string` |
|------|---------------------|---------------|
| **大小** | 固定，必须指定 | 动态，自动调整 |
| **安全性** | 缓冲区溢出风险 | 安全，自动边界检查 |
| **内存** | 手动管理 | 自动（RAII） |
| **操作** | C 函数（strcpy、strcat） | 丰富的成员函数 |
| **比较** | strcmp 函数 | ==、!=、< 运算符 |
| **连接** | strcat（不安全） | + 运算符 |

### 11.2.8 String 与 Vector<char> 的对比

`string` 和 `vector<char>` 类似，但 `string` 具有额外的字符串专用功能：

| 特性 | `string` | `vector<char>` |
|------|----------|----------------|
| **用途** | 文本数据 | 通用字符数组 |
| **连接** | `+` 运算符 | 无 `+` 运算符 |
| **比较** | `==`、`<` 等 | `==` 检查内容，`<` 不标准 |
| **C 字符串转换** | `c_str()`、`data()` | 需手动转换 |
| **空终止符** | 自动 | 不自动 |
| **流 I/O** | `>>`、<<` | 不直接支持 |

> **文本处理用 `string`**。**仅在需要不带字符串语义的原始字符数组时**使用 `vector<char>`。

### 11.3.2 排序算法

| 算法 | 说明 | 示例 | 时间复杂度 |
|------|------|------|------------|
| `sort(begin, end)` | 升序排序 | `sort(v.begin(), v.end())` | O(n log n) |
| `sort(begin, end, comp)` | 自定义比较器排序 | `sort(v.begin(), v.end(), greater<int>())` | O(n log n) |
| `stable_sort(begin, end)` | 稳定排序 | `stable_sort(v.begin(), v.end())` | O(n log² n) 或 O(n log n) |
| `partial_sort(begin, mid, end)` | 部分排序 | `partial_sort(v.begin(), v.begin()+5, v.end())` | O(n log n) |
| `nth_element(begin, nth, end)` | 以第 n 个元素为界划分 | `nth_element(v.begin(), v.begin()+5, v.end())` | 平均 O(n) |

**示例：**
```cpp
vector<int> v = {3, 1, 4, 1, 5, 9, 2, 6};

// 升序排序
sort(v.begin(), v.end());
// v = {1, 1, 2, 3, 4, 5, 6, 9}

// 降序排序
sort(v.begin(), v.end(), greater<int>());
// v = {9, 6, 5, 4, 3, 2, 1, 1}

// 自定义比较器（按绝对值排序）
sort(v.begin(), v.end(), [](int a, int b) {
    return abs(a) < abs(b);
});

// 仅排序前 5 个元素
partial_sort(v.begin(), v.begin()+5, v.end());

// 找中位数（划分后中间元素）
nth_element(v.begin(), v.begin() + v.size()/2, v.end());
int median = v[v.size()/2];
```

### 11.3.3 查找算法

| 算法 | 说明 | 示例 | 时间复杂度 |
|------|------|------|------------|
| `find(begin, end, val)` | 线性查找 | `find(v.begin(), v.end(), 5)` | O(n) |
| `binary_search(begin, end, val)` | 二分查找（仅有序区间） | `binary_search(v.begin(), v.end(), 5)` | O(log n) |
| `lower_bound(begin, end, val)` | 首个 >= val 的元素 | `lower_bound(v.begin(), v.end(), 5)` | O(log n) |
| `upper_bound(begin, end, val)` | 首个 > val 的元素 | `upper_bound(v.begin(), v.end(), 5)` | O(log n) |
| `equal_range(begin, end, val)` | 等于 val 的元素区间 | `equal_range(v.begin(), v.end(), 5)` | O(log n) |
| `find_if(begin, end, pred)` | 按谓词查找 | `find_if(v.begin(), v.end(), isEven)` | O(n) |

**示例：**
```cpp
vector<int> v = {1, 2, 3, 4, 5, 6, 7, 8, 9};

// 线性查找
auto it = find(v.begin(), v.end(), 5);
if (it != v.end()) {
    cout << "在索引 " << (it - v.begin()) << " 找到" << endl;
}

// 二分查找（要求区间有序）
bool exists = binary_search(v.begin(), v.end(), 5);  // true

// 下界/上界
auto lb = lower_bound(v.begin(), v.end(), 5);  // 指向 5
auto ub = upper_bound(v.begin(), v.end(), 5);  // 指向 6

// 统计出现次数
int count = ub - lb;  // 1（vector 中只有一个 5）

// 查找第一个偶数
auto it2 = find_if(v.begin(), v.end(), [](int x) {
    return x % 2 == 0;
});
// it2 指向 2
```

### 11.3.4 最值算法


### 11.3.5 修改型算法

| 算法 | 说明 | 示例 | 备注 |
|------|------|------|------|
| `copy(begin, end, dest)` | 拷贝区间 | `copy(v1.begin(), v1.end(), v2.begin())` | 目标必须有足够空间 |
| `copy_if(begin, end, dest, pred)` | 条件拷贝 | `copy_if(v.begin(), v.end(), dest, isEven)` | C++11 |
| `fill(begin, end, val)` | 用值填充 | `fill(v.begin(), v.end(), 0)` | |
| `generate(begin, end, gen)` | 用生成器填充 | `generate(v.begin(), v.end(), rand)` | |
| `replace(begin, end, old, new)` | 替换值 | `replace(v.begin(), v.end(), 1, 99)` | |
| `replace_if(begin, end, pred, new)` | 条件替换 | `replace_if(v.begin(), v.end(), isOdd, 0)` | |
| `reverse(begin, end)` | 反转顺序 | `reverse(v.begin(), v.end())` | |
| `rotate(begin, mid, end)` | 旋转元素 | `rotate(v.begin(), v.begin()+3, v.end())` | |
| `shuffle(begin, end, rng)` | 随机打乱 | `shuffle(v.begin(), v.end(), rng)` | C++11 |
| `unique(begin, end)` | 移除连续重复项 | `unique(v.begin(), v.end())` | 之后需调用 erase |

**示例：**
```cpp
vector<int> v = {1, 2, 3, 4, 5};
vector<int> dest(5);

// 拷贝
copy(v.begin(), v.end(), dest.begin());
// dest = {1, 2, 3, 4, 5}

// 仅拷贝偶数
copy_if(v.begin(), v.end(), dest.begin(), [](int x) {
    return x % 2 == 0;
});
// dest[0] = 2, dest[1] = 4，其余未指定（未调整大小时）

// 填充
fill(v.begin(), v.end(), 0);  // v = {0, 0, 0, 0, 0}

// 生成
int n = 0;
generate(v.begin(), v.end(), [&n]() { return n++; });
// v = {0, 1, 2, 3, 4}

// 替换
replace(v.begin(), v.end(), 3, 99);  // v = {0, 1, 2, 99, 4}

// 反转
reverse(v.begin(), v.end());  // v = {4, 99, 2, 1, 0}

// remove-erase 惯用法（remove 不会真正调整大小）
v = {1, 2, 3, 2, 4, 2, 5};
auto newEnd = remove(v.begin(), v.end(), 2);  // 将非 2 元素前移
v.erase(newEnd, v.end());  // 真正移除尾部元素
// v = {1, 3, 4, 5}

// 移除连续重复项
v = {1, 1, 2, 2, 2, 3, 3};
auto it = unique(v.begin(), v.end());  // 返回新的尾后迭代器
v.erase(it, v.end());
// v = {1, 2, 3}
```

### 11.3.9 分区算法

| 算法 | 说明 | 示例 |
|------|------|------|
| `partition(begin, end, pred)` | 按谓词分区 | `partition(v.begin(), v.end(), isEven)` |
| `stable_partition(begin, end, pred)` | 稳定分区 | 保持相对顺序 |
| `is_partitioned(begin, end, pred)` | 检查是否已分区 | `is_partitioned(v.begin(), v.end(), isEven)` |
| `partition_point(begin, end, pred)` | 查找分界点 | 谓词首次为假的迭代器 |

**示例：**
```cpp
vector<int> v = {1, 2, 3, 4, 5, 6, 7, 8, 9};

// 分区：偶数在前，奇数在后
auto it = partition(v.begin(), v.end(), [](int x) {
    return x % 2 == 0;
});
// v = {2, 4, 6, 8, 1, 3, 5, 7, 9}（顺序可能不同）
// it 指向第一个奇数元素（1）

// 稳定分区保持相对顺序
v = {1, 2, 3, 4, 5, 6, 7, 8, 9};
stable_partition(v.begin(), v.end(), [](int x) {
    return x % 2 == 0;
});
// v = {2, 4, 6, 8, 1, 3, 5, 7, 9}（偶数和奇数保持原序）
```

### 11.3.10 堆算法

| 算法 | 说明 | 示例 |
|------|------|------|
| `make_heap(begin, end)` | 创建大根堆 | `make_heap(v.begin(), v.end())` |
| `push_heap(begin, end)` | 向堆中添加元素 | push_back 后调用 push_heap |
| `pop_heap(begin, end)` | 将堆顶移至末尾 | pop_heap 后调用 pop_back |
| `sort_heap(begin, end)` | 堆排序 | `sort_heap(v.begin(), v.end())` |
| `is_heap(begin, end)` | 检查是否为堆 | `is_heap(v.begin(), v.end())` |

**示例：**
```cpp
vector<int> v = {3, 1, 4, 1, 5, 9, 2, 6};

// 创建堆
make_heap(v.begin(), v.end());
// v 现在是一个大根堆：最大元素在 v[0]

// 访问最大值
int max = v.front();  // 9

// 提取最大值
pop_heap(v.begin(), v.end());  // 将最大值移到末尾
v.pop_back();  // 移除最大值

// 添加新元素
v.push_back(10);
push_heap(v.begin(), v.end());  // 恢复堆性质

// 堆排序
make_heap(v.begin(), v.end());
sort_heap(v.begin(), v.end());  // 升序排序
```

### 11.3.11 最佳实践

**1. 正确使用迭代器：**
```cpp
// 好：适用于任何容器
sort(v.begin(), v.end());

// 灵活性较差：仅适用于 vector
sort(v.data(), v.data() + v.size());
```

**2. 检查返回值：**
```cpp
auto it = find(v.begin(), v.end(), 42);
if (it != v.end()) {
    // 找到
} else {
    // 未找到
}
```

**3. 使用 lambda 实现自定义行为：**
```cpp
// 按绝对值降序排序
sort(v.begin(), v.end(), [](int a, int b) {
    return abs(a) > abs(b);
});

// 查找第一个长度大于 5 的字符串
auto it = find_if(strings.begin(), strings.end(), [](const string& s) {
    return s.length() > 5;
});
```

**4. 对数组使用 std::begin() 和 std::end()：**
```cpp
int arr[] = {3, 1, 4, 1, 5};
sort(begin(arr), end(arr));  // C++11，兼容原生数组
```

**5. 拷贝前预留空间：**
```cpp
vector<int> dest;
dest.reserve(src.size());  // 避免重新分配
copy(src.begin(), src.end(), back_inserter(dest));
```

**6. 使用算法代替手写循环：**
```cpp
// 避免：
int sum = 0;
for (int x : v) sum += x;

// 使用：
int sum = accumulate(v.begin(), v.end(), 0);
```
