# 1 命令行基础

Windows 命令提示符 (CMD) 是一个命令行解释器，允许用户使用文本命令与操作系统交互。

## 1.1 打开命令提示符

### 1.1.1 打开 CMD 的方法

| 方法 | 步骤 |
|------|------|
| **运行对话框** | 按 `Win + R`，输入 `cmd`，按 Enter |
| **开始菜单** | 搜索 "命令提示符" 或 "cmd" |
| **文件资源管理器** | 在地址栏输入 `cmd`，按 Enter |
| **右键菜单** | Shift + 右键点击文件夹 → "在此处打开命令窗口" |

### 1.1.2 管理员权限

某些命令需要提升的权限：

1. 右键点击"命令提示符"
2. 选择"以管理员身份运行"
3. 确认 UAC 提示

> **注意：** 管理员 CMD 在标题栏显示 "管理员:"。

## 1.2 命令语法

### 1.2.1 基本结构

```
命令 [选项] [参数]
```

| 组件 | 描述 | 示例 |
|------|------|------|
| **命令** | 要执行的操作 | `dir`, `copy`, `del` |
| **选项** | 修改命令行为 | `/s`, `/q`, `/y` |
| **参数** | 命令的目标 | `file.txt`, `C:\Users` |

### 1.2.2 常用选项

| 选项 | 含义 | 用法 |
|------|------|------|
| `/?` | 显示帮助 | `dir /?` |
| `/s` | 包含子目录 | `dir /s` |
| `/q` | 安静模式（无提示） | `del /q *.tmp` |
| `/y` | 对提示回答是 | `copy /y file.txt` |
| `/a` | 显示隐藏文件 | `dir /a` |

### 1.2.3 获取帮助

```cmd
:: 显示命令帮助
命令 /?

:: 示例
dir /?
copy /?
mkdir /?
```

## 1.3 文件路径

### 1.3.1 路径类型

| 类型 | 描述 | 示例 |
|------|------|------|
| **绝对路径** | 从根目录开始的完整路径 | `C:\Users\John\Documents` |
| **相对路径** | 从当前目录开始的路径 | `Documents\file.txt` |

### 1.3.2 特殊字符

| 字符 | 含义 | 示例 |
|------|------|------|
| `\` | 目录分隔符 | `C:\Windows\System32` |
| `..` | 上级目录 | `cd ..` |
| `.` | 当前目录 | `copy .\file.txt D:\` |
| `*` | 通配符（任意字符） | `del *.txt` |
| `?` | 通配符（单个字符） | `file?.txt` |
| `""` | 引号（用于路径中的空格） | `cd "Program Files"` |

### 1.3.3 路径示例

```cmd
:: 绝对路径
cd C:\Users\John\Documents

:: 相对路径（向上一级）
cd ..\Pictures

:: 使用通配符
dir *.txt
doc*.doc
file?.txt

:: 带空格的路径（必须用引号）
cd "C:\Program Files\Java"
copy "my document.txt" D:\Backup\
```

# 2 导航命令

导航命令允许您在文件系统中移动。

## 2.1 cd（切换目录）

`cd` 命令更改当前工作目录。

### 2.1.1 语法

```cmd
cd [盘符:][路径]
cd [/d] [盘符:][路径]
```

### 2.1.2 基本用法

| 命令 | 描述 |
|------|------|
| `cd Documents` | 进入 "Documents" 子目录 |
| `cd ..` | 返回上级目录 |
| `cd ..\..` | 向上返回两级 |
| `cd \` | 返回磁盘根目录 |
| `cd /d D:\` | 切换到 D: 盘 |

### 2.1.3 示例

```cmd
:: 导航到特定文件夹
cd C:\Users\John\Documents

:: 向上导航一级
cd ..

:: 导航到根目录
cd \

:: 切换磁盘（使用 /d 开关）
cd /d D:\Games

:: 直接输入盘符也可以
D:
cd Games
```

## 2.2 dir（目录列表）

`dir` 命令显示目录的内容。

### 2.2.1 语法

```cmd
dir [盘符:][路径][文件名] [/a[[:]<属性>]] [/b] [/s] [/w]
```

### 2.2.2 常用选项

| 选项 | 描述 |
|------|------|
| `/a` | 显示指定属性的文件 |
| `/b` | 简洁格式（仅文件名） |
| `/s` | 包含子目录 |
| `/w` | 宽格式（多列） |
| `/o` | 按名称排序 |
| `/t:w` | 按修改时间排序 |

### 2.2.3 示例

```cmd
:: 基本列表
dir

:: 列出所有文件包括隐藏文件
dir /a

:: 包含子目录列表
dir /s

:: 简洁格式（用于批处理）
dir /b

:: 宽格式
dir /w

:: 按日期排序
dir /t:w /o:-d

:: 仅列出 .txt 文件
dir *.txt

:: 列出特定目录中的文件
dir C:\Windows\System32\*.dll
```

## 2.3 tree（目录结构）

tree 命令以树状格式显示目录结构。

### 2.3.1 语法

```cmd
tree [盘符:][路径] [/f] [/a]
```

### 2.3.2 选项

| 选项 | 描述 |
|------|------|
| `/f` | 显示每个目录中的文件名 |
| `/a` | 使用 ASCII 字符而不是扩展字符 |

### 2.3.3 示例

```cmd
:: 显示目录树
tree

:: 显示包含文件的树
tree /f

:: 将树保存到文件
tree /f > structure.txt
```

# 3 文件操作

文件操作命令用于创建、查看、复制、移动和删除文件。

## 3.1 type（显示文件内容）

`type` 命令显示文本文件的内容。

### 3.1.1 语法

```cmd
type [盘符:][路径]文件名
```

### 3.1.2 示例

```cmd
:: 显示文件内容
type readme.txt

:: 使用 more 分页显示
type longfile.txt | more

:: 连接多个文件
type file1.txt file2.txt > combined.txt
```

> **注意：** `type` 用于文本文件。对于二进制文件，使用 `more` 或 `findstr`。

## 3.2 copy（复制文件）

`copy` 命令复制一个或多个文件。

### 3.2.1 语法

```cmd
copy [/y | /-y] 源文件 目标文件
copy 源文件1 + 源文件2 目标文件
```

### 3.2.2 选项

| 选项 | 描述 |
|------|------|
| `/y` | 覆盖时不提示确认 |
| `/-y` | 覆盖时提示确认 |
| `/v` | 验证文件写入是否正确 |

### 3.2.3 示例

```cmd
:: 复制单个文件
copy file.txt D:\Backup\

:: 复制并重命名
copy file.txt D:\Backup\backup.txt

:: 复制多个文件
copy *.txt D:\Backup\

:: 复制并改名
copy report.txt "2024-03-25_report.txt"

:: 连接文件
copy file1.txt + file2.txt combined.txt

:: 强制覆盖不提示
copy /y file.txt D:\Backup\

:: 从其他位置复制到当前目录
copy C:\Users\John\Desktop\file.txt .
```

## 3.3 move（移动/重命名文件）

`move` 命令移动文件或重命名文件。

### 3.3.1 语法

```cmd
move [/y | /-y] 源文件 目标文件
```

### 3.3.2 示例

```cmd
:: 将文件移动到另一个文件夹
move file.txt D:\Archive\

:: 重命名文件
move oldname.txt newname.txt

:: 移动多个文件
move *.log D:\Logs\

:: 无确认移动
move /y file.txt D:\Destination\
```

> **注意：** `move` 默认会覆盖现有文件而不警告。使用 `/-y` 进行提示。

## 3.4 del（删除文件）

`del` 命令删除一个或多个文件。

### 3.4.1 语法

```cmd
del [/p] [/f] [/s] [/q] [/a[[:]<属性>]] 文件名
```

### 3.4.2 选项

| 选项 | 描述 |
|------|------|
| `/p` | 删除前提示确认 |
| `/f` | 强制删除只读文件 |
| `/s` | 从子目录中删除 |
| `/q` | 安静模式（无确认） |
| `/a` | 删除指定属性的文件 |

### 3.4.3 示例

```cmd
:: 删除单个文件
del file.txt

:: 带确认删除
del /p file.txt

:: 删除多个文件
del *.tmp

:: 强制删除只读文件
del /f readonly.txt

:: 从子目录中删除
del /s *.bak

:: 安静删除（无提示）
del /q *.log

:: 删除隐藏文件
del /a:h hidden.txt
```

> **警告：** 删除的文件不会进入回收站。它们被永久删除！

## 3.5 ren（重命名文件）

`ren`（rename）命令重命名文件。

### 3.5.1 语法

```cmd
ren [盘符:][路径]文件名1 文件名2
```

### 3.5.2 示例

```cmd
:: 重命名单个文件
ren old.txt new.txt

:: 使用通配符重命名
ren *.txt *.bak

:: 在特定目录中重命名
ren C:\Temp\file.txt newname.txt
```

# 4 目录操作

目录操作命令用于创建和删除目录。

## 4.1 mkdir/md（创建目录）

`mkdir`（或 `md`）命令创建新目录。

### 4.1.1 语法

```cmd
mkdir [盘符:]路径
md [盘符:]路径
```

### 4.1.2 示例

```cmd
:: 创建单个目录
mkdir NewFolder

:: 创建嵌套目录
mkdir C:\Users\John\Documents\Projects\2024

:: 同时创建多个目录
mkdir Folder1 Folder2 Folder3

:: 创建带空格的路径（加引号）
mkdir "C:\My Projects\New Folder"
```

## 4.2 rmdir/rd（删除目录）

`rmdir`（或 `rd`）命令删除目录。

### 4.2.1 语法

```cmd
rmdir [/s] [/q] [盘符:]路径
rd [/s] [/q] [盘符:]路径
```

### 4.2.2 选项

| 选项 | 描述 |
|------|------|
| `/s` | 删除目录及其所有内容（子目录和文件） |
| `/q` | 安静模式（无确认） |

### 4.2.3 示例

```cmd
:: 删除空目录
rmdir EmptyFolder

:: 删除有内容的目录
rmdir /s FolderWithContents

:: 强制删除无确认
rmdir /s /q OldFolder

:: 多个目录
rmdir Folder1 Folder2
```

> **警告：** `/s /q` 会永久删除所有内容。请非常小心！

# 6 网络命令

网络命令用于诊断和显示网络配置。

## 6.1 ipconfig（IP 配置）

`ipconfig` 命令显示网络配置。

### 6.1.1 语法

```cmd
ipconfig [/all] [/release] [/renew] [/flushdns]
```

### 6.1.2 选项

| 选项 | 描述 |
|------|------|
| `/all` | 显示详细配置 |
| `/release` | 释放 IP 地址（DHCP） |
| `/renew` | 续订 IP 地址（DHCP） |
| `/flushdns` | 清除 DNS 缓存 |
| `/displaydns` | 显示 DNS 缓存 |

### 6.1.3 示例

```cmd
:: 基本 IP 配置
ipconfig

:: 详细配置
ipconfig /all

:: 释放和续订 IP
ipconfig /release
ipconfig /renew

:: 清除 DNS 缓存
ipconfig /flushdns
```

## 6.2 ping（测试连通性）

`ping` 命令测试与主机的网络连通性。

### 6.2.1 语法

```cmd
ping [-t] [-n count] [-l size] 目标
```

### 6.2.2 选项

| 选项 | 描述 |
|------|------|
| `-t` | 持续 ping（按 Ctrl+C 停止） |
| `-n count` | ping 次数 |
| `-l size` | 数据包大小（字节） |
| `-a` | 将地址解析为主机名 |

### 6.2.3 示例

```cmd
:: 默认 ping（4 次）
ping google.com

:: ping 指定次数
ping -n 10 8.8.8.8

:: 持续 ping
ping -t google.com

:: 指定数据包大小 ping
ping -l 1024 google.com

:: ping 本地主机（测试本地 TCP/IP）
ping 127.0.0.1
```

## 6.3 tracert（跟踪路由）

`tracert` 命令跟踪到目标的路由。

### 6.3.1 语法

```cmd
tracert [-d] [-h maximum_hops] 目标
```

### 6.3.2 示例

```cmd
:: 跟踪到 Google 的路由
tracert google.com

:: 最多 30 跳
tracert -h 30 example.com

:: 不解析主机名（更快）
tracert -d 8.8.8.8
```

## 6.4 netstat（网络统计）

`netstat` 命令显示网络连接。

### 6.4.1 语法

```cmd
netstat [-a] [-b] [-n] [-o] [-p protocol]
```

### 6.4.2 选项

| 选项 | 描述 |
|------|------|
| `-a` | 显示所有连接和监听端口 |
| `-b` | 显示可执行文件名 |
| `-n` | 以数字形式显示地址 |
| `-o` | 显示进程 ID |
| `-p` | 显示特定协议的连接 |

### 6.4.3 示例

```cmd
:: 显示活动连接
netstat

:: 所有连接及进程 ID
netstat -ano

:: 按协议显示
netstat -p tcp

:: 显示可执行文件名（需要管理员权限）
netstat -b
```

## 6.5 其他网络命令

| 命令 | 描述 | 示例 |
|------|------|------|
| `nslookup` | 查询 DNS | `nslookup google.com` |
| `route` | 查看/编辑路由表 | `route print` |
| `arp` | 显示 ARP 缓存 | `arp -a` |
| `telnet` | 测试端口连通性 | `telnet host port` |
| `getmac` | 显示 MAC 地址 | `getmac` |

# 7 高级主题

## 7.1 重定向操作符

重定向操作符控制输入/输出流。

| 操作符 | 描述 | 示例 |
|--------|------|------|
| `>` | 将输出重定向到文件（覆盖） | `dir > files.txt` |
| `>>` | 将输出重定向到文件（追加） | `echo line >> file.txt` |
| `<` | 从文件重定向输入 | `command < input.txt` |
| `|` | 将输出管道传递给另一个命令 | `dir | more` |
| `2>` | 重定向错误 | `command 2> errors.txt` |

### 7.1.1 示例

```cmd
:: 将目录列表保存到文件
dir > files.txt

:: 追加到文件
echo New line >> log.txt

:: 管道传递给 more（分页输出）
dir | more

:: 管道传递给 find
dir | find "txt"

:: 重定向错误
command 2> error.log

:: 重定向输出和错误
command > output.txt 2>&1
```

## 7.2 批处理文件基础

批处理文件（.bat）包含多个 CMD 命令。

### 7.2.1 基本结构

```batch
@echo off
echo Hello World
echo Current date: %DATE%
pause
```

### 7.2.2 常用批处理命令

| 命令 | 描述 |
|------|------|
| `@echo off` | 不显示命令 |
| `echo.` | 空行 |
| `pause` | 等待按键 |
| `goto` | 跳转到标签 |
| `if` | 条件执行 |
| `for` | 遍历项目 |

### 7.2.3 简单批处理文件示例

```batch
@echo off
echo ===================================
echo        系统信息报告
echo ===================================
echo.
echo 计算机: %COMPUTERNAME%
echo 用户:     %USERNAME%
echo 日期:     %DATE% %TIME%
echo.
echo --- 网络信息 ---
ipconfig | find "IPv4"
echo.
echo --- 磁盘空间 ---
wmic logicaldisk get size,freespace,caption
echo.
pause
```

## 7.3 实用快捷键

| 快捷键 | 操作 |
|--------|------|
| `Tab` | 自动补全文件/文件夹名 |
| `↑ / ↓` | 循环浏览命令历史 |
