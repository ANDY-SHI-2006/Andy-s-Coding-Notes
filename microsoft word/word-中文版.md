## 第0章 基础知识和快捷键

### 文件操作

| 操作 | 快捷键/方法 | 说明 |
|------|------------|------|
| 快速缩放 | `Ctrl` + 滚轮 | 调整文档显示比例 |
| 创建新文档 | `Ctrl` + `N` | 新建空白 Word 文档 |
| 另存为 | - | 可修改格式解决不同版本兼容性问题 |

### 编辑快捷键

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 撤销键入 | `Ctrl` + `Z` | 撤销上一步操作 |
| 恢复键入 | `Ctrl` + `Y` | 恢复被撤销的操作 |
| 重复上一步操作 | `F4` 或 `Alt` + `Enter` | 复制一次后可以粘贴多次，也可以连用 `F4` |

> 💡 **提示**：利用剪贴板可以实现多次复制，统一粘贴（剪贴板中内容会自下而上粘贴）

### 对齐快捷键

| 对齐方式 | 快捷键 |
|---------|--------|
| 居中 | `Ctrl` + `E` |
| 左对齐 | `Ctrl` + `L` |
| 右对齐 | `Ctrl` + `R` |

### Tab 键的功能

- **段首空两格**：一个 `Tab` 等于两个空格
- **批量 Tab**：选中多行文字后按 `Tab` 键
- **反向 Tab**：`Shift` + `Tab`
- **表格中的妙用**：
  - 按下 `Tab` 键向右移一格
  - 在右下角格按 `Tab` 自动加一行
- **快速切换文档**：`Ctrl` + `Tab`

### 选中技巧

| 选中目标 | 操作方法 |
|---------|---------|
| 选中一个词 | 在文本中双击 |
| 选中一句话 | 先将光标放在对应的话上，再 `Ctrl` + 单击 |
| 选中一行 | 在文本框左侧单击 |
| 选中一段 | 在文本框左侧双击，或在文本中三击 |
| 连续选取 | 起始点用光标左键锁定，按住 `Shift` 键后单击 |
| 竖向选择 | 按住 `Alt` 键，鼠标左键拖选 |
| 加选（不连续） | 先选中一个目标后，按住 `Ctrl` 加选 |
| 选择格式相似的文本 | ①左键选中对应文本 ②"开始"选项卡中找到"选择"图标 ③选择"选择格式相似的文本" |
| 选中全文 | 在文本框左侧三击，或 `Ctrl` + `A` |

### 特殊按键

| 按键 | 功能 |
|------|------|
| `Backspace` | 向左删除 |
| `Del` | 向右删除 |
| `Home` | 回到行首 |
| `End` | 回到行尾 |
| `PgUp` / `PgDn` | 向上/下翻（相当于窗口大小的一页，并非文档中的一页） |
| `Insert` (`Ins`) | 按下后新输入的字会替换后面的字 |

## 第1章 开始选项卡

### 输入文字与格式刷

**格式刷使用方法**：
1. 选中所需复制的格式，按格式刷，再选中需处理的文本（建议从右往左刷）
2. 选中原文字后 `Ctrl` + `Shift` + `C`，再选中待处理文字 `Ctrl` + `Shift` + `V`

**连刷**：
- 选中后双击格式刷再连刷
- 取消连刷：再次单击格式刷 / 单击 `ESC`

**F4 重复操作**：更改后再选中其他文字，再按 `F4` 可实现类似格式刷的操作

**清除格式**：
- 上方橡皮符号
- 格式刷

### WPS 堆叠功能

在 WPS 中堆叠在一起的功能：
1. 加拼音
2. 更改大小写（重要功能）
3. 加圈
4. 加框

### 更改大小写

适合多段批量处理英文文本

### 加拼音

1. 先进行改字号等操作，最后加拼音（否则会错乱）
2. 仅保留拼音：选中后右键，点击切换域代码，仅删除对应汉字即可获得仅含拼音的字符

### 样式

1. 应用系统已有的样式
2. 新建自己的样式并可随时修改（可实现超远程联动修改）

- 可隐藏页间空白

### 纸张大小和纸张方向

1. 可以调整纸张类型

### 分栏

1. 可以实现形似《植物生理学》的页面布局
